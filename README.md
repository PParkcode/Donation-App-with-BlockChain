# Donation-App-with-BlockChain
### 블록체인을 이용한 기부 플랫폼입니다.
원화와 1대1 가치를 가지는 토큰을 통해 기부가 진행되며, 기부를 비롯한 모든 토큰들의 이동은 블록에 기록되며, 누구도 수정할 수 없습니다.<br>
